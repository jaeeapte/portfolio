Overview

As a part of "Information Visualization" class, we designed new visualizations for structures like Temporal, Hierarchical, Network, Text and Storytelling each week. For these exercises we broke into groups and used a new technique to ideate for possible visualizations. We each create domain questions that the viz should answer, go through through 3 rounds of ideating, eliminating design ideas in each round, till we reach a group consensus on a final design.


Tennis viewer

Create a list of domain information for a tennis match viewer and design a visualization to support those domain questions.


Each person on the group first created domain questions in addition to existing domain question cards. We then place all the cards on table and pick 4 most crucial ones in the order of importance.



Each person then sketches at least 3 visualizations that answer the domain questions which is followed by a final iteration of a sketch.



The group then evaluates merits and shortcomings of each design against the domain questions and decides one design.
