Problem

The problem is the existing communication gap between the speech - impaired people, who use the Indian Sign Language (ISL)) and the others. The current solutions available are neither affordable or intuitive.

Solution

We developed an application to recognize the gestures of Indian Sign Language (ISL) and present the English translation of those gestures in text form, to aid in effective communication between the speech-impaired people and the others using assistive technology.

    • Calculating Hidden Markov Model (HMM)

    Different people have different ways of performing the same gesture. We decided to use a probabilistic model to store gesture information to overcome this hurdle and improve the accuracy significantly. The Hidden Markov Model is a stable form of storing probabilities. Hence, we decided to implement HMM in our system. Understanding HMM was a crucial part to our success. We decided to recognize 5 gestures, namely, girl, curtain, open, drive , car. Each of these signs had some movement from an initial position to a final position, which we identified as ‘states’. We collected training sets for each gesture and calculated the HMM for each.


    Each HMM consists of 3 matrices, namely, π which is the initial state probability, A which is the state transition probability and B which is the observation probability. Using the training data for each class, we computed the corresponding HMM λ(A, B, π)


    For each gesture to be recognized by the system, we generate a training data set of observations
    O = (O1, O2, …, OL). We use these observations to learn λ.
    In our application,
    • For each pre-defined λ, we compute P (O|λ), the Probability of Occurrence of O given the model.
    • For recognizing a gesture, we estimate the appropriate λ, which maximizes P (O|λ).

    We created HMMs of all signs.
    • When a user performs a gesture, an observation sequence comprising 3 states is generated.
    • The probability of this sequence is checked against pre defined HMMs
    • The one with the highest probability is recognised as the performed gesture.



    We created a space zoning matrix to extrapolate the raw data provided by Kinect's skeletal tracker to identify joints from both the hands.

    Take Away

    On completion of the first working prototype, we found over 90% accurate results for the sample set of gestures.
